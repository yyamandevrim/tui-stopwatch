# Changelog

All notable changes to this project are documented in this file.

## 1.1.0 - 2026-04-07

### Added
- Theme presets: classic, neon, amber, mono.
- Runtime theme cycling with the t key.
- Split list navigation with up/down, j/k, pageup/pagedown, home/end.

### Changed
- Configuration now supports a top-level theme value with color overrides.
- Splits pane now supports viewport-aware scrolling and selected row highlighting.
- README updated with theme and split navigation documentation.
